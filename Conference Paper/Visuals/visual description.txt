VISUAL DESCRIPTION FILE
========================
This file contains descriptions of all visualizations in the Visuals folder.

EXPLORATORY DATA ANALYSIS (EDA) VISUALS
========================================

1. EDA_target_distribution.png
   Description: Target Variable Distribution Analysis showing four panels:
   (a) Histogram of corn production showing right skew
   (b) Box plot revealing outliers
   (c) Log-transformed distribution showing normalization
   (d) Q-Q plot confirming approximate normality after log transformation
   This visualization demonstrates why log transformation is necessary for the target variable.

2. EDA_temporal_analysis.png
   Description: Temporal Analysis of Corn Production showing four panels:
   (a) Mean production over time with standard deviation bands
   (b) Production variance showing increasing variability
   (c) Number of observations per year
   (d) Total state production over time
   This visualization reveals temporal trends and patterns in corn production across the 23-year study period.

3. EDA_correlation_target.png
   Description: Feature Correlation with Target Variable showing:
   (a) Top 15 positive correlations with corn production
   (b) Top 15 negative correlations with corn production
   This visualization identifies which features are most strongly associated with the target variable.

4. EDA_correlation_heatmap.png
   Description: Correlation Heatmap for Top 20 Features showing correlation strength between features.
   Color intensity represents correlation strength, with red indicating positive correlations and blue indicating negative correlations.
   This visualization reveals feature clusters and multicollinearity patterns.

5. correlation_heatmap.png
   Description: Alternative correlation heatmap visualization showing relationships between features.
   Similar to EDA_correlation_heatmap.png but may use different feature selection or visualization style.

6. EDA_PCA_analysis.png
   Description: PCA Analysis Results showing four panels:
   (a) Scree plot showing explained variance by component
   (b) Cumulative explained variance with 90% and 95% thresholds
   (c) PC1 vs PC2 scatter plot colored by target variable
   (d) Top 10 feature loadings for PC1 and PC2
   This visualization demonstrates dimensionality reduction opportunities and principal component interpretation.

7. EDA_KNN_analysis.png
   Description: K-Nearest Neighbors Visualization showing query points (red stars) and their k nearest neighbors (blue circles) visualized in 2D PCA space.
   Multiple panels show k = 5, 10, 20, 50 neighbors.
   Gray points represent all data samples.
   This visualization reveals data structure and local similarity patterns.

8. EDA_KNN_distances.png
   Description: Average Distance to k-th Nearest Neighbor showing how neighbor distances increase with k.
   This indicates data density and local structure in the high-dimensional feature space.

9. EDA_feature_distributions.png
   Description: Distribution Analysis for Top 12 Features showing histograms of feature distributions with mean and median lines.
   Features are selected by correlation with target variable or variance.
   This visualization provides insights into data characteristics across different feature types.

10. EDA_missing_values.png
    Description: Missing Value Analysis showing the top 10 columns with missing data.
    Displays the number of missing values per column.
    This visualization helps identify data completeness issues and guides imputation strategy.

11. EDA_outlier_detection.png
    Description: Outlier Detection Analysis showing box plots for top 6 features with outliers.
    Upper and lower bounds (3×IQR) shown as dashed red lines.
    This visualization identifies extreme values that may require special handling.

12. temporal_analysis.png
    Description: Alternative temporal analysis visualization showing time-based patterns in the data.
    May focus on different temporal aspects or use different visualization style than EDA_temporal_analysis.png.
